# E-Commerce Sales Analysis

## Project Overview

This project analyzes an E-Commerce sales dataset using Python and Power BI to identify sales trends, customer behavior, and business insights.
The project includes:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Interactive Power BI Dashboard
